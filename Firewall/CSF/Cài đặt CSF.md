# Bước 3: Tải CSF
```
    cd /tmp
    wget https://download.configserver.com/csf.tgz
```
![image](https://user-images.githubusercontent.com/110179869/195227809-5ee2766d-bdf0-4b80-98c6-d055e297ca00.png)

# Bước 4: Giải nén và cài đặt
```
    tar -xzf csf.tgz
    cd csf
    sh install.sh
```
